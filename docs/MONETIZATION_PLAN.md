## 1. 앱 현황 분석

### 핵심 가치
- **개인 독서 관리**: 책 등록, 독서 상태 추적, 히스토리 관리
- **다양한 책 등록 방식**: 알라딘 검색, 바코드 스캔, 직접 입력
- **레트로 UI**: 픽셀아트 스타일의 차별화된 디자인 (DungGeunMo 폰트)

### 수익화에 유리한 기존 자산
- 알라딘 API 키(TTB_KEY) 이미 보유 → 제휴 연동 기반 확보
- 소셜 로그인 3종 완비 → 사용자 식별/구독 결제 기반 확보
- 독서 데이터 축적 구조 → 통계/인사이트 서비스 확장 가능
- 레트로 디자인 컨셉 → 굿즈/내보내기 콘텐츠와 잘 어울림

## 2. 단기 전략 (구현 난이도 낮음)

### 2-1. 알라딘 제휴 링크

**개요:** 책 검색/상세 화면에서 알라딘 구매 페이지로 연결. 구매 발생 시 제휴 수수료 수익.

**구현 범위:**
- `BookInfoScreen`에 "알라딘에서 구매하기" 버튼 추가 (이미 "알라딘에서 보기" 버튼 존재)
- `SearchBookScreen` 검색 결과 아이템에 구매 링크 연결
- 제휴 링크 URL에 TTB_KEY 파라미터 포함

**예상 작업량:** 1~2일

**수익 구조:**
- 알라딘 파트너 프로그램 가입 필요
- 구매 건당 일정 비율 수수료 (보통 1~3%)
- 유저가 이미 책을 검색하는 동선이 있으므로 전환율 기대

**장점:**
- 이미 TTB_KEY를 보유하고 있어 기술 연동이 거의 완료된 상태
- 사용자 경험을 해치지 않는 자연스러운 동선
- 추가 UI 요소 최소화

**유의사항:**
- 알라딘 파트너 약관 확인 필요
- 앱 내 직접 결제가 아니므로 Google Play 수수료 대상 아님

---

### 2-2. 프리미엄 구독 (월정액) — 기본형

**개요:** 무료 사용자에게 기능 제한을 두고, 구독자에게 무제한 + 추가 기능 제공.

**Free vs Premium 구분:**

| 기능 | Free | Premium |
|------|------|---------|
| 책 등록 | 최대 20권 | 무제한 |
| 바코드 스캔 | 일 3회 | 무제한 |
| 히스토리 조회 | 최근 10권 | 전체 |
| 독서 통계 | X | O |
| 내보내기 | X | O |
| 광고 | 있음 | 없음 |

**구현 범위:**
- Google Play Billing Library 연동
- 구독 상태 관리 (서버 또는 로컬)
- 기능 제한 로직 추가

**예상 작업량:** 1~2주

**가격 제안:** 월 1,900원 / 연 15,900원

## 3. 중기 전략 (구현 난이도 중간)

### 3-1. 독서 통계 / 인사이트 (프리미엄 킬러 피처)

**개요:** 사용자의 독서 데이터를 분석하여 시각적 통계와 인사이트 제공.

**제공 통계:**
- 월별/연별 독서량 (읽은 권 수, 총 페이지 수)
- 장르별 분포 차트
- 평균 독서 기간 (시작일 → 종료일)
- 독서 연속 기록 (streak)
- "올해 가장 빨리 읽은 책", "가장 오래 읽은 책" 등 흥미 요소

**구현 범위:**
- 백엔드: 통계 API 엔드포인트 추가
- 프론트: 통계 화면 (차트 라이브러리 또는 Canvas 커스텀)
- 하단 네비게이션에 "통계" 탭 추가 또는 설정 내 메뉴

**예상 작업량:** 2~3주

**수익 연계:** 프리미엄 구독의 핵심 가치로 포지셔닝

---

### 3-2. 독서 목표 / 챌린지

**개요:** "이번 달 N권 읽기" 같은 목표 설정. 달성 시 뱃지/보상.

**기능:**
- 월간/연간 독서 목표 설정
- 진행률 표시 (프로그레스 바)
- 달성 뱃지 컬렉션 (레트로 픽셀아트 뱃지)
- 목표 달성 시 축하 화면

**수익 연계:**
- 뱃지 컬렉션 → 프리미엄 전용
- 광고 보고 힌트/보상 얻기 (보상형 광고)
- 게이미피케이션으로 DAU/MAU 증가 → 광고 수익 증대

**예상 작업량:** 2~3주

---

### 3-3. 비파괴적 네이티브 광고

**개요:** 홈 피드 하단 또는 검색 결과 사이에 네이티브 광고 삽입.

**광고 위치 후보:**
- 홈 화면(내 서점) 하단 배너
- 검색 결과 목록 사이 네이티브 광고 (매 10개 결과마다)
- 히스토리 화면 하단

**구현:**
- Google AdMob SDK 연동
- 레트로 UI 톤에 맞는 광고 스타일링 (가능한 범위 내)
- 프리미엄 구독자는 광고 제거

**유의사항:**
- 레트로 UI의 특색을 해치지 않도록 광고 배치 신중하게
- 사용자 경험 우선 — 전면 광고는 지양

**예상 작업량:** 1주

## 4. 장기 전략 (구현 난이도 높음)

### 4-1. 독서 커뮤니티 / 리뷰

**개요:** 같은 책을 읽는 유저끼리 짧은 감상을 공유하는 소셜 기능.

**기능:**
- 책 상세 화면에 한 줄 감상 남기기
- 같은 책을 읽은 다른 사용자의 감상 보기
- 좋아요 / 공감 기능
- 인기 감상 피드

**수익 연계:**
- 네트워크 효과로 유저 유입 → 광고/프리미엄 기반 수익 확대
- 커뮤니티 기능 자체를 프리미엄으로 제한 가능

**고려사항:**
- 백엔드 인프라 확장 필요 (감상 저장, 알림 등)
- 콘텐츠 모더레이션 필요
- 사용자 수가 일정 규모 이상이어야 의미 있음

---

### 4-2. 출판사 / 서점 협업 (B2B)

**개요:** 신간 추천, 이벤트 배너, 프로모션 등 B2B 광고.

**형태:**
- "이번 주 신간" 추천 섹션 (출판사 협찬)
- 특정 서점 이벤트 배너
- 북큐레이션 컨텐츠 (출판사 제공)

**전제 조건:**
- 월간 활성 사용자(MAU) 일정 규모 확보 필요
- 독서 관련 앱으로서의 브랜드 인지도 구축

---

### 4-3. 독서 기록 내보내기 (유료)

**개요:** 연간 독서 기록을 예쁜 PDF/이미지로 내보내기.

**컨셉:**
- 레트로 픽셀아트 스타일의 독서 카드/포스터 생성
- "나의 2026년 독서 리포트" — 연간 통계 + 읽은 책 목록 + 인사이트
- SNS 공유용 이미지 (인스타그램 스토리 사이즈)
- PDF 다운로드

**수익 연계:**
- 기본 내보내기는 무료, 프리미엄 템플릿은 유료
- 연말 시즌에 특히 수요가 높을 것으로 예상
- 레트로 UI 컨셉과 잘 어울리는 "굿즈" 느낌

**예상 작업량:** 3~4주

## 5. 추천 로드맵

```
Phase 1 (즉시)
  └─ 알라딘 제휴 링크 적용
       ↓
Phase 2 (1~2개월)
  ├─ 독서 통계 기능 개발
  ├─ AdMob 네이티브 광고 삽입
  └─ 프리미엄 구독 도입
       ↓
Phase 3 (3~4개월)
  ├─ 독서 목표/챌린지
  └─ 독서 기록 내보내기
       ↓
Phase 4 (6개월+)
  ├─ 독서 커뮤니티
  └─ B2B 협업
```

### 우선순위 근거

1. **알라딘 제휴 링크** — 이미 API 키가 있고 검색 동선이 완성되어 있어 가장 빠르게 적용 가능. 리스크 없음.
2. **독서 통계 + 프리미엄 구독** — 앱의 핵심 가치(독서 관리)와 직결. 통계가 프리미엄의 킬러 피처 역할.
3. **AdMob 광고** — 무료 사용자 기반 수익. 프리미엄 업셀 유도 효과.
4. **독서 목표/내보내기** — 리텐션 강화 + 추가 유료 기능. 통계와 시너지.
5. **커뮤니티/B2B** — 사용자 규모 확보 후 진행.
